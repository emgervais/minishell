PWD et OLDPWD ne doivent pas etre recrées si elles ont été supprimées par la commande unset
Example : 
```
minishell$ env | grep PWD
PWD=/Users/ele-sage/Desktop/minishell
minishell$ unset PWD
minishell$ env | grep PWD
minishell$
minishell$ cd ..
minishell$ env | grep PWD
OLDPWD=/Users/ele-sage/Desktop
PWD=/Users/ele-sage
cd ..
minishell$ env | grep PWD
OLDPWD=/Users/ele-sage/Desktop
PWD=/Users/ele-sage

vs

bash-3.2$ env | grep PWD
PWD=/Users/ele-sage/Desktop/minishell
bash-3.2$ unset PWD
bash-3.2$ env | grep PWD
bash-3.2$
bash-3.2$ cd ..
bash-3.2$ env | grep PWD
OLDPWD=
bash-3.2$ cd ..
bash-3.2$ env | grep PWD
OLDPWD=/Users/ele-sage/Desktop
```
